
# 计算机网络知识体系交互式展示系统

## 项目简介

本项目是一个集知识展示、原理解析、可视化呈现、交互学习于一体的计算机网络原理交互式 Web 系统，包含前端、后端、数据库三个完整层次。涵盖三大模块：单项协议原理可视化动态展示、综合网络场景模拟、TCP/IP 五层知识体系管理与知识图谱。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | 原生 HTML5 / CSS3 / JavaScript（Canvas 动画）、D3.js v7（知识图谱） |
| 后端 | Python Flask |
| 数据库 | SQLite |

零 npm 依赖，无需 Node.js 环境。

## 环境配置

```powershell
pip install flask
```

## 运行方式

```powershell
cd D:\network-interactive-system
python app.py
```

浏览器访问 `http://127.0.0.1:5000`

## 主要功能

### 模块一：单项协议原理可视化动态展示
- **DNS 域名解析**：Canvas 动画展示从客户端到根 DNS、顶级 DNS、权威 DNS 的完整递归查询过程，含缓存表和最终 IP 结果
- **TCP 三次握手与四次挥手**：Canvas 动画逐步展示 SYN/ACK/FIN 报文传递过程及客户端/服务器状态变化

### 模块二：综合网络场景模拟
- 给定拓扑下（H1→交换机→路由器→Internet→Web 服务器）的 8 步通信过程逐步拆解
- 每步展示：步骤名称、起点/终点、协议类型、广播/单播、源/目的 MAC/IP、传输路径高亮、交换表/ARP 表变化

### 模块三：计算机网络知识体系综合展示与交互学习
- TCP/IP 五层模型（应用层/传输层/网络层/数据链路层/物理层）知识点管理
- 支持增删改查、模糊搜索、分页
- 知识图谱：D3.js 力导向图展示各层协议与技术的层次关系
